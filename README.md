# Nexus

Nexus is a lightweight productivity desktop app.

### Features
* note taking
* task management
* snippets
* timer
* RSS features
---

**Quick Start**

1. Install dependencies

```cmd
npm install
```

2. Run development mode (Vite + Electron)

```cmd
npm run dev
```

3. Build for production

```cmd
npm run build
npm run build:electron
```
